# Autobazar - OOP projekt

Jednoduchá webová aplikace pro správu autobazaru, vytvořená v JS s OOP

## Funkce

- Použití LocalStorage
- Přidávání aut přes formulář
- Zobrazení seznamu všech aut
- Mazání aut ze seznamu
- Zobrazení souhrnu "statistik"
- Filtrace aut ze seznamu

## Spuštění
1. Stáhněte nebo klonujte repo
2. Otevřete `index.html` v prohlížeči
3. Hotovo :D
